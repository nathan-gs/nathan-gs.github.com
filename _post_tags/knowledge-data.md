title: Knowledge & Data
---

Knowledge & Data covers the foundation everything else is built on: how organisations store, process, model, and ultimately *understand* their data. It spans two decades of this blog — from distributed processing on Hadoop and Spark, through real-time and streaming architectures, to decentralised ownership with Data Mesh, and on to the semantic layer that agentic systems now depend on.

The through-line is that the technology keeps changing and the problem doesn't. Getting the right data to the right place has always been the hard part. Agents made it urgent rather than new: an [AI OS](/tags/ai-os) hits the context wall for the same reason a query hits a full table scan — the working set was wrong.
